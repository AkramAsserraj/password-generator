# Secure Password Generator

Application Python permettant de générer et tester des mots de passe sécurisés.

---

## Fonctionnalités

- Génération de mots de passe sécurisés
- Choix des caractères (majuscules, minuscules, chiffres, symboles)
- Vérification de la sécurité d’un mot de passe
- Historique des mots de passe (version GUI)
- Deux versions disponibles : Interface graphique (Tkinter) et Terminal (CLI)

---

### Versions disponibles

## 1. Interface graphique (Tkinter)
- Interface simple et intuitive
- Boutons et options interactives

## 2. Version terminal (CLI)
- Utilisation via menu dans le terminal
- Léger et rapide

---

#### Technologies utilisées

- Python
- Tkinter
